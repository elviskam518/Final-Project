Project Summary

This demo presents an academic pipeline for studying and mitigating bias in hiring decisions. It reuses the existing project code for fairness metrics (b.py) and baseline/adversarial modeling (c.py), while exposing Fair CVAE variants through archived experiment outputs.

  • Baseline MLP
  • Standalone adversarial baseline (from c.py)
  • Fair CVAE adv_only, no_adv, and full

Use Results for precomputed research outputs and Interactive Demo for upload → analysis → model execution workflow.
